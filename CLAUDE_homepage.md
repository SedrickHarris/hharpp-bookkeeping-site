# Hharpp Homepage — Claude Code Project Brief

## Task
Build a single production-ready HTML file: `index.html`
No other files. No external CSS. No external JS. Everything inline.

## Constraints
- Self-contained: one file, deployable as-is
- All CSS in one `<style>` block in `<head>`
- All JS in one `<script>` block before `</body>`
- Google Fonts via `@import` inside `<style>`
- All 5 schema blocks as `<script type="application/ld+json">` in `<head>`
- Responsive: 375px / 768px / 1280px breakpoints
- No broken image tags — use styled `<div>` placeholders
- No dead `#` hrefs on navigation links
- WCAG AA color contrast on all text
- Validate: run the file through a browser before marking done

## File to produce
`/hharpp-homepage/index.html`
